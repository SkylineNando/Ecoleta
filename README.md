## Getting Started

To start the project you need to install NodeJs, React, React Native and Sqlite3.

### Installing

#### Run server: 

```npm run dev```

#### Create migrates:

```knex:migrate```

#### Create seeders:

```knex:seed```

#### Run application WEB:

```yarn start```

##### or 

```npm start```

#### Run application mobile:

```expo start```
